== Description ==

Blog to pdf allows your users to download or print any section of your Wordpress as a PDF document. 

The pro version additionally offers:

* Page numbers
* Auto generation of an indexed table of content
* Set a title for the front page
* Set an image to the front page
* Choose the Wordpress theme used to style the PDF

== Installation ==

1. Upload the `blogtopdf` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
